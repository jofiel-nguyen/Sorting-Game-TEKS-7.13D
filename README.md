# 🧬 TEKS 7.13D: Selection Sorting Challenge

A fast-paced, 5-minute interactive classroom game designed to help 7th-grade students differentiate between **Natural Selection** and **Selective Breeding (Artificial Selection)**.

---

## 🎯 Educational Objective
**TEKS 7.13D:** The student is expected to differentiate between inherited and acquired physical traits of organisms and describe the role of internal and external factors in the process of natural selection and selective breeding.

---

## 🛠️ How to Set Up
1. **Create a Folder:** Create a new folder on your computer (e.g., "SelectionGame").
2. **Save the Files:** Place the following three files inside that folder:
   * `index.html`
   * `style.css`
   * `script.js`
3. **Launch:** Double-click the `index.html` file to open the game in your web browser.

---

## 🎮 How to Play
1. **The Goal:** Drag all the trait cards from the "Item Bank" into the correct category before the time runs out.
2. **The Timer:** You have **3 minutes** (180 seconds). The clock starts as soon as you drag the first item.
3. **Validation:** * **Correct:** The item will stay in the bucket and show a green checkmark.
   * **Incorrect:** You will see a red "Try Again" message, and you must move the item to the other bucket.
4. **End of Game:** * If the bank is cleared, you win!
   * If time runs out, the game freezes for a class-wide review.

---

## ⚙️ Teacher Customization
To add more examples or change the terms, open `script.js` in a text editor (like Notepad) and look for the `items` list at the top. 

**Format to add a new item:**
`{ id: '9', text: 'Your New Trait Here', category: 'natural' },`

* Use `category: 'natural'` for items controlled by nature.
* Use `category: 'selective'` for items controlled by humans.
